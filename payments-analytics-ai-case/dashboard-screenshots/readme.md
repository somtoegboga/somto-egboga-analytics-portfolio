# About the screenshots in this folder

1. digital wallet rewards overview; is the executive level insight into the success/failure of the program. showing that while users in this program spent on average almost $2M a month, the program was consistently operating on a loss of around $10K per month. as we had a negative marketing roi, there needs to be reconsideration into the fee or cashback structure
2. loss from failure view: shows which pathway failure most occurs on to enhance data decision on technical trouble shooting.
3. customer segment & opportunity: is a responsive dashboard to use the monetary value of the customer segments to see which product categories are most popular within the group. this should help with designing more tailored rewards program by customer segment.

Tool used; Tableau Desktop (coonnected to our transformed fact and dimensions table from Snowflake)
