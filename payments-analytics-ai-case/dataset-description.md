# Dataset Description
The dataset includes the following features:

1. idx: Unique index for each record
2. transaction_id:Unique identifier for each transaction (UUID)
3. user_id: Unique identifier for each user
4. transaction_date: Date and time of the transaction
5. product_category: Category of the product or service
6. product_name: Specific product or service name
7. merchant_name: Name of the merchant or service provider
8. product_amount: Transaction amount in local currency
9. transaction_fee: Fee charged for the transaction
10. cashback: Cashback amount received for the transaction
11. loyalty_points: Loyalty points earned from the transaction
12. payment_method: Method used for payment
13. transaction_status: Status of the transaction (Successful, Failed, Pending)
14. merchant_id: Unique identifier for each merchant
15. device_type: Type of device used for the transaction
16. location: Broad location category of the transaction


Digital Wallets Dataset courtesy of [Kaggle](https://www.kaggle.com/datasets/harunrai/digital-wallet-transactions?resource=download)
